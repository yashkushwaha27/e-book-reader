A Quiet Demo

This is a bundled HTML book. Each top-level section in the body becomes one spread-friendly page in the reader.

Chapter II — Turning

Use the Next page control (or drag a corner) to move forward with the page-flip animation powered by StPageFlip.

Chapter III — Your shelf

Add entries in src/data/books.ts. Point htmlUrl at static files under public/books/ or at any URL that allows your origin to fetch HTML.

Chapter IV — Google Drive

For Drive-hosted HTML, set gdriveFileId on a catalog row. While you run yarn dev, Vite proxies that file so the reader can load it without browser CORS blocking the request.
